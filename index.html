Đam mê xây dựng các giải pháp công nghệ giải quyết bài toán kinh doanh thực tế — từ tối ưu cơ sở dữ liệu, thiết kế API sạch, giao diện web hiện đại đến hệ thống hiệu năng cao.

Kinh nghiệm: Xây dựng hệ thống quản lý đa phòng ban, dashboard real-time, API sync tự động, ETL pipeline và các workflow automation phức tạp.
